# UE5 Course Practice 🎮

This repository contains my Unreal Engine 5 C++ practice project.

## 🚀 Features

* Custom Actors (DestructibleActor, Treasure, Weapon)
* Character system (Hero, BaseCharacter)
* Enemy logic
* Health system with UI (HealthBar, AttributeComponent)
* Interfaces for interaction (HitInterface)

## 🧠 What I learned

* Unreal Engine C++ architecture
* Actor & Component system
* Character movement & animation setup
* Basic combat & interaction systems

## 📁 Project Structure

* `Source/` → All C++ game logic
* `Config/` → Engine configuration files

## ⚠️ Note

This repository contains **code only**.
Assets (textures, models, etc.) are excluded for optimization.

Made with Unreal Engine 5.4
